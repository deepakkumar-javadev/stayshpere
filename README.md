# StaySphere(Project)

StaySphere is a backend system designed to handle hotel bookings, room management, and customer reservations in a simple and efficient way. It focuses on providing a smooth experience for managing hotel operations through REST APIs.

## About the Project

This project is built to simulate a real-world hotel management system where users can browse rooms, make bookings, and manage their reservations. On the backend side, it handles all core operations like room availability, booking flow, and customer data management.

The goal of this project is to demonstrate strong backend development skills using Java and Spring Boot, along with proper database design and RESTful API development.

## Key Features

. Room management (add, update, delete rooms)
. Customer registration and management
. Room booking and cancellation
. Check room availability
. Booking history tracking

## Tech Stack

. Java
. Spring Boot
. Hibernate / JPA
. MySQL
. REST APIs
. Maven

## Project Purpose

This project was created as a learning and practice project to strengthen backend development skills and understand how real-world booking systems work behind the scenes.
